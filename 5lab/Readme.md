# Решение нелинейных уравнений и систем

Лабораторная работа по численным методам.

## Структура проекта

```
├── first_part.py          # Задание 1: решение нелинейного уравнения
├── 2part.py               # Задание 2: решение системы нелинейных уравнений
├── input.py               # Входные параметры
├── run_env.sh             # Активация виртуального окружения
├── execute_me.sh          # Запуск обоих заданий
├── requirements.txt       # Зависимости
└── README.md              # Этот файл
```

## Задание 1

Решение нелинейного уравнения \( f(x) = 0 \) тремя методами:

| Метод | Сходимость |
|-------|-----------|
| Дихотомия | Всегда сходится |
| Простая итерация | \( \|\phi'(x)\| < 1 \) |
| Ньютона | \( f'(x) \neq 0 \) |


## Задание 2

Решение системы нелинейных уравнений:

\[
\begin{cases}
f_1(x, y) = 0 \\
f_2(x, y) = 0
\end{cases}
\]

Методы:
- Ньютона (с матрицей Якоби)
- Простой итерации
- Зейделя

Начальное приближение — точка пересечения графиков.

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd <папка-проекта>
```

### 2. Создание виртуального окружения

```bash
python3 -m venv venv
```

### 3. Активация окружения

```bash
source venv/bin/activate        # Linux/Mac
# или
venv\Scripts\activate           # Windows
```

### 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 5. Настройка параметров

Отредактируйте `input.py`:

```python
# Задание 1
f_input = "x**3 - 3*x + 1"   # ваше уравнение
eps = 0.001                   # точность
left = -3                     # левая граница
right = 3                     # правая граница

# Задание 2
f1_system = "x**2 + y**2 - 4"   # первое уравнение
f2_system = "x - y**2"          # второе уравнение
x0_system = 1.5                  # начальное x
y0_system = 1.2                  # начальное y
```

### 6. Запуск

```bash
# Активировать окружение
source venv/bin/activate

# Запустить задание 1
python3 first_part.py

# Запустить задание 2
python3 2part.py

# Или запустить оба сразу
./execute_me.sh
```
